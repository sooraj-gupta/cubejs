Add a 3D loading cube to any website

Sooraj Gupta Decemeber 19, 2020

Add a rotating cube to any website for a loading screen as seen on this website itself:

  • Download cube.js
  • Add to website
  • Customize settings

    • Download cube.js

    Download the zipped file then move the unzipped folder to your website folder.

       Download


    Add to Website

    Add the cube.css file to the end of the <head> section

    <link href = "cube/cube.css" rel = "stylesheet">

    Add settings.js and cube.js to the beginning of the<body> section in this order

    <script src = "cube/settings.js"></script>
<script src = "cube/cube.js"></script>

    Customize Settings

    Change the duration and other settings ( to be added later ) in the settings.js file 

    var settings = 
{
	//Set to false to disable loading
	"load": true,
	
	//Loading time in rotations
	"rotations": 3
}

    Change the color and size of the cube in the cube.css file.

    :root
{
	--size: calc( 5vw + 50px );
	--shadowcolor:#d09f9f;
	--lightcolor: #ffd3d3;
}

    The --size variable represents the width of each face of the cube.

    The --lightcolor variable represents the color of the face that is facing forwards.

    The --shadowcolor variable represents the color of all the other faces giving a "lighting" effect
